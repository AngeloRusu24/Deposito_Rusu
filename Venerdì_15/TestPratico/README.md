# Fiori & Petali

Sito web per un fiorario locale. Pagina singola con HTML e CSS puri, nessun framework.

## File

```
TestPratico/
index.html
style.css
README.md
```

## Sezioni

- **Header** — logo + navigazione con anchor link
- **Hero** — titolo, sottotitolo e bottone
- **Prodotti** — griglia di 3 card con CSS Grid
- **Contatti** — form con nome, email, messaggio
- **Footer** — indirizzo e copyright

## Tecnologie

- HTML5 semantico
- CSS3 (Flexbox, Grid, hover, transizioni)
- Nessuna libreria esterna
